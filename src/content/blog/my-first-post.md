# 欢迎来到我的技术世界！

今天，我正式使用 **Astro** 搭建了我的技术博客。

## 为什么选择 Astro？

- ⚡ 极快加载速度（纯 HTML + 按需 JS）
- ✍️ 用 Markdown 写文章，简单高效
- 🚀 自动部署到 Vercel，无需服务器运维
- 🎨 支持自定义样式，可打造 Apple 风格界面

- 学习 JavaScript 和 React
- 尝试构建一个小型工具
- 用博客记录每个项目
